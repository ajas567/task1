# Simple Interest Calculator

This project is a simple calculator that computes the simple interest given the principal amount, annual rate of interest, and time period.

## Features

* Calculates simple interest
* Takes principal, rate, and time as inputs
* Outputs the calculated simple interest

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Technologies Used

* Bash Shell Script

## How to Run

1. Clone the repository:

```
git clone https://github.com/yourusername/simple-interest-calculator.git
```

2. Navigate to the project directory:

```
cd simple-interest-calculator
```

3. Run the script:

```
bash simple-interest.sh
```

## Example

If:

* Principal = 1000
* Rate = 5%
* Time = 2 years

Simple Interest = 100
